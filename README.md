# Ground Station Alpha Theme

A modern, dark-mode-first Jekyll theme designed for technical blogs, specifically optimized for satellite communications and radio monitoring content.

## Features

- 🌓 **Dark/Light Mode** - Automatic switching with localStorage persistence
- 🔍 **Real-time Search** - Fast client-side search with keyboard shortcuts (Cmd+K)
- 📡 **Technical Aesthetic** - Satellite/radio themed with animated signal indicators
- 📱 **Fully Responsive** - Optimized for all screen sizes
- 🎯 **Table of Contents** - Auto-generated with scroll spy
- 📋 **Code Copy Buttons** - One-click code copying
- 📊 **Reading Progress Bar** - Visual indicator while scrolling
- 🏷️ **Categories & Tags** - Organized content browsing
- ⚡ **Fast Performance** - Minimal dependencies, optimized CSS
- 🔒 **SEO Ready** - Built-in SEO tags and structured data

## Installation

### 1. Clone or Download

```bash
git clone https://github.com/hmdsamarasekara/hmdsamarasekara.github.io.git
cd hmdsamarasekara.github.io
```

### 2. Install Dependencies

```bash
bundle install
```

### 3. Configure

Edit `_config.yml` with your information.

### 4. Run Locally

```bash
bundle exec jekyll serve
```

Visit `http://localhost:4000` to preview.

### 5. Deploy to GitHub Pages

Push to your repository and enable GitHub Pages in settings.

## Creating Content

### New Post

Create a file in `_posts/` with format: `YYYY-MM-DD-title.md`

```yaml
---
layout: post
title: "Your Post Title"
date: 2023-01-01 12:00:00 +0530
categories: [Category1, Category2]
tags: [tag1, tag2, tag3]
excerpt: "Brief description for cards and search"
---

Your content here...
```
