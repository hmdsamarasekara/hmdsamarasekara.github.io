# Noise Floor Nomad

A clean, dark-mode Jekyll theme for ham radio and SDR enthusiasts.

## Features

- Dark/Light mode toggle
- Animated spectrum analyzer display
- Real-time search
- Responsive design
- QRZ contact integration

## Installation

1. Replace your repo files with these
2. Add your logo as `assets/images/logo.png`
3. Update posts with front matter
4. Push to GitHub
